# Сводка внесенных изменений в проект Apliko Frontend

## Добавленные функции

### 1. Пользовательское соглашение на странице регистрации

**Файлы изменены:**
- `src/page/login/ui/signup/signup.tsx` - добавлена ссылка на пользовательское соглашение
- `src/shared/i18n/resource/ru/auth.json` - добавлен перевод для "пользовательского соглашения"

**Новые файлы:**
- `src/page/terms-of-service/index.tsx` - компонент страницы пользовательского соглашения
- `src/shared/i18n/resource/ru/main.json` - добавлены переводы для пользовательского соглашения

**Функциональность:**
- Флажок согласия теперь включает ссылку на пользовательское соглашение
- Ссылка открывается в новой вкладке (`/terms-of-service`)
- Полный текст соглашения содержит 6 разделов с юридически корректным содержанием
- Поддержка интернационализации через систему переводов

### 2. Информация об организации на главной странице

**Файлы изменены:**
- `src/page/main/main.tsx` - добавлен новый компонент OrganizationInfo
- `src/shared/i18n/resource/ru/main.json` - добавлена информация об организации

**Новые файлы:**
- `src/page/main/ui/organization-info/organization-info.tsx` - компонент с информацией об организации
- `src/page/main/ui/organization-info/index.ts` - экспорт компонента

**Функциональность:**
- Отображение официальной информации об ООО "Аплико"
- Данные о деятельности в сфере разработки ПО
- Информация о резидентстве в ИТ-парке
- Упоминание о включении в региональный реестр стартапов РТ
- Юридические данные: ИНН, ОГРН, адрес, дата регистрации
- Визуальные иконки для ключевых достижений

## Совместимость

Все изменения совместимы с существующей архитектурой проекта:
- Используется существующая система интернационализации (react-i18next)
- Применяются существующие стили и компоненты UI
- Сохранена структура папок и соглашения по именованию
- Добавленные компоненты следуют паттернам проекта

## Следующие шаги

Для полной интеграции изменений необходимо:
1. Добавить маршрут для `/terms-of-service` в роутер приложения
2. Протестировать отображение на различных устройствах
3. Проверить корректность переводов
4. При необходимости добавить английские переводы
