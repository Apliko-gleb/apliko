# APLIKO Frontend

## Локальный запуск

Версия Node.js **v23.3.0**

### Установка зависимостей

```
yarn
```

### Запуск в дев-режиме

```
yarn dev
```

Открывать по адресу **http://localhost:3000**

### Сборка проекта 

```
yarn build
```

### Предпросмотр собранного проекта

```
yarn preview
```

Открывать по адресу **http://localhost:3030**

## Запуск с помощью docker-контейнера

### Билд образа

```
docker build -t image_name .
```

### Запуск контейнера

```
docker run -d -p 3000:80 image_name
```

Открывать по адресу **http://localhost:3000**
